Presentation

This software has been developped by Alan Guitard, student in Computer Science at Bordeaux University, under the supervision of Norbert Mercier and Loïc Martin, in the laboratory IRAMAT-CRP2A. It has been made to be used with the DosiVox2d software. This interface allows to create a voxelized world with, besides, a 2D render of that object. So, you can easily manipulate the world, build your project and generates pilot text files (PTF) to run the simulation in DosiVox2d.

Just few things you have to know about the interface manipulation

  • When you save the project, you does not generate the Pilot Text File.
  • In the "Project information" area, press ENTER has the effect to validate values. The current fields stays red-bordered. But don't worry, the field is updated.
